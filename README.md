# 🇧🇩 Liberation War 1971 Website

A simple and responsive website based on the **Bangladesh Liberation War of 1971**.
This project highlights the history, heroes, and memories of the মহান মুক্তিযুদ্ধ.

---

## 🚀 Live Preview

👉 https://liberation-war-1971.vercel.app 

---

## 📌 Features

* Responsive design using Bootstrap
* Clean and simple UI
* Navigation bar with multiple pages
* Hero section with introduction
* History overview section
* Footer section

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5

## 📖 About the Project

The **Liberation War of 1971** was a significant event in Bangladesh's history.
This website is created to present:

* Historical background
* Important events
* Freedom fighters (heroes)
* Visual gallery
